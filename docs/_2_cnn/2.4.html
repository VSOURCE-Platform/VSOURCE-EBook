一、说明

首先要知道，AlexNet是2012年提出来了，到今天有了9年的历史，这个所谓是“第一个卷积神经网络”，但要注意，这个并不是第一个“深度学习”网络，因为AlexNet的网络层数很浅，并没有拉伸。要注意，在这篇AlexNet的论文中，也是GPU第一次被用于训练神经网络。

其原论文目前被引量接近8万，可以直接通过scholar.google查阅到原文：

AlexNet的名称也是来源于第一作者的名称，这也是第一次采用卷积神经网络的方法来提取图像里和标签对应的特征。要注意本质上神经网络在提取特征，提取的特征指的是显著性与提供的标签所对应的图像的特征。

作为深度学习的开山鼻祖，这篇文章在条件允许的情况下，还是希望大家能够读一下。

二、卷积神经网络部件（Components）

先不谈AlexNet的细节，对于一般的简单的卷积神经网络的组成，如下图所示：

可以看到，一共有五个组成部分：

  1. 输入层（Input Layer）：往往指的是输入的图像，这里往往要固定好图像的shape，即height, width, channel的形式。
import tensorflow as tf

def model(shape):
    input_layer = tf.keras.layers.Input(shape)
    
    model = tf.keras.models.Model(inputs=input_layer, outputs=input_layer)
    return model

if __name__ == '__main__':
    a = model([224,224,3])
    print(a)
  1. 卷积层（Convolutional Layer）：即在之前的课程中，我们所讲述过的卷积提取图像特征的操作，这里是可以进行加深拉伸的，并不是只有一层就可以。 原图->卷积层，卷积层输出得到的是特征图，对特征图，依然可以接卷积层，进行更深度的特征的提取。也就是我们很容易看到 原图->卷积层->卷积层->卷积层这样的操作。
import tensorflow as tf

def model(shape):
    input_layer = tf.keras.layers.Input(shape)
    feature_map1 = tf.keras.layers.Conv2D(filters=64, kernel_size=[3,3], padding='same')(input_layer)
    print(feature_map1)
    
    model = tf.keras.models.Model(inputs=input_layer, outputs=feature_map1)
    return model

if __name__ == '__main__':
    a = model([224,224,3])
    print(a)

几个重要的参数： ​

（1）filters：卷积核数量，数量越多希望提取的特征越多，会作为输出矩阵的channel。

（2）kernel_size：卷积核大小，越大感受也越大，特征越宏观。

（3）strides：步长，默认为1，设置为2的时候矩阵shape会缩减一半。

（4）padding：是否加0来补偿卷积核移动的时候造成的shape的破坏，默认为valid不补偿，设置为same的时候在步长为1的情况下，输入输出的shape不会发生变化。

（5）其他...

  1. 池化层（Pooling Layer）：池化是不带参数的，这里往往指的是对于一个 h,w,c的shape的特征图，进行放缩，比如[1024，1024，3]的原图经过多次卷积层后，shape依然很大，会导致计算量很大，所以这里我们可以用池化，将特征图放缩一半，比如从[512,512,64]的特征图shape放缩到[256,256,64]这样。
import tensorflow as tf

def model(shape):
    input_layer = tf.keras.layers.Input(shape)
    feature_map1 = tf.keras.layers.Conv2D(filters=64, kernel_size=[3,3], padding='same')(input_layer)
    feature_map2 = tf.keras.layers.Conv2D(filters=64, kernel_size=[3,3], padding='same')(feature_map1)

    pool1 = tf.keras.layers.MaxPool2D(pool_size=(2,2), padding='same')(feature_map2)
	print(pool1)
    
    model = tf.keras.models.Model(inputs=input_layer, outputs=pool1)
    return model

if __name__ == '__main__':
    a = model([224,224,3])
    print(a)

几个重要的参数：

（1）pool_size：池化层窗口大小，要池化的窗口的大小，对Max池化则保留窗口内最大的值，对Avg池化则对窗口内的值求平均。 （2）strides：池化窗口的步长，类似卷积的步长。 （3）padding：类似卷积的padding。 一般来说，pool_size为[3,3]，strides为[2,2]，padding为same的配置是最常见的，这样的配置下输出的shape相对于输入shape产生了一次缩减。

  1. 全连接层（Fully Connected Layer）：简称FC，这里的全连接层主要实际上指的是向量间的线性变换加一个非线性激活函数的过程，是无论机器学习还是深度学习中都非常重要的方法，可以实现向量维度的变换，增强线性能力和非线性能力的方式。而往往如果解决的是图像分类问题，最后一层FC的激活函数往往是Softmax。
import tensorflow as tf

def model(shape):
    input_layer = tf.keras.layers.Input(shape)
    feature_map1 = tf.keras.layers.Conv2D(filters=64, kernel_size=[3,3], padding='same')(input_layer)
    feature_map2 = tf.keras.layers.Conv2D(filters=64, kernel_size=[3,3], padding='same')(feature_map1)

    pool1 = tf.keras.layers.MaxPool2D(pool_size=(2,2), padding='valid')(feature_map2)
    features = tf.keras.layers.GlobalAveragePooling2D()(pool1)
	print(features)
    
    fc1 = tf.keras.layers.Dense(units=512, activation='relu')(features)
    fc2 = tf.keras.layers.Dense(units=10, activation='softmax')(fc1)
    
    model = tf.keras.models.Model(inputs=input_layer, outputs=fc2)
    return model

if __name__ == '__main__':
    a = model([224,224,3])
    print(a)

几个重要的参数： （1）units：FC全连接神经元的数量，越多计算量越大。 （2）activation：激活函数的指定。

三、AlexNet结构

结构是从下到上，最下方的输入图像的shape为[224,224,3]，整体流程:

  1. 经过一个kernel 11x11的，strides为4的卷积层，输出shape为[56,56,96]。
  2. 经过LRN层，这个实际上在现代已经被废弃了，输出shape为[56,56,96]。
  3. 经过MaxPool层，输出shape为[28,28,96]。
  4. 经过一个卷积层，输出shape为[28,28,256]。
  5. 经过LRN层。
  6. 经过MaxPool层，输出shape为[14,14,96]。
  7. 经过三个卷积层，输出shape为[14,14,256]。
  8. 经过MaxPool层，输出shape为[7,7,256]。
  9. 矩阵拉平Flatten()，得到7x7x256长度的向量，经过一个FC，得到4096的向量，relu激活。
  10. 继续经过一个FC，得到一个4096的向量，relu激活。
  11. 最终经过一个1000（类别数）的FC，softmax激活。

整体可以用TensorFlow翻译为：

import tensorflow as tf

def alexnet(shape):
    input_layer = tf.keras.layers.Input(shape)
    
    x = tf.keras.layers.Conv2D(filters=96, kernel_size=[11,11], strides=[4,4], padding='same', activation='relu')(input_layer)
    x = tf.keras.layers.LayerNormalization()(x)
    x = tf.keras.layers.MaxPool2D(pool_size=(3,3), strides=[2,2], padding='same')(x)
    
    x = tf.keras.layers.Conv2D(filters=256, kernel_size=[5,5], padding='same', activation='relu')(x)
    x = tf.keras.layers.LayerNormalization()(x)
    x = tf.keras.layers.MaxPool2D(pool_size=(3,3), strides=[2,2], padding='same')(x)
    
    x = tf.keras.layers.Conv2D(filters=384, kernel_size=[3,3], padding='same', activation='relu')(x)
    x = tf.keras.layers.Conv2D(filters=384, kernel_size=[3,3], padding='same', activation='relu')(x)
    x = tf.keras.layers.Conv2D(filters=256, kernel_size=[3,3], padding='same', activation='relu')(x)
    
    x = tf.keras.layers.MaxPool2D(pool_size=(3,3), strides=[2,2], padding='same')(x)
    
    x = tf.keras.layers.Flatten()(x)
    x = tf.keras.layers.Dense(units=4096, activation='relu')(x)
    x = tf.keras.layers.Dense(units=4096, activation='relu')(x)
    x = tf.keras.layers.Dense(units=1000, activation='softmax')(x)
    
    model = tf.keras.models.Model(inputs=input_layer, outputs=x)
    return model

if __name__ == '__main__':
    a = alexnet([224,224,3])
    print(a)
